📑 Автоматический Белый Список (Whitelist) для VPN

Репозиторий содержит актуальные списки доменов и IP-адресов, которые должны направляться в обход VPN (прямое подключение). Списки генерируются автоматически и обновляются каждый час, что гарантирует актуальность всех адресов.🚀 Особенности проекта

🔄 Частое обновление: Скрипты проверяют и обновляют данные каждый час (24/7).

⚡ Высокая скорость: Оптимизированные форматы для быстрой загрузки клиентами.

🤖 Полный автомат: Списки собираются без ручного вмешательства через GitHub Actions.

🛠 Как использовать (Инструкция)Для подключения списка в ваше приложение, используйте прямую ссылку (Raw) на нужный файл.
1. Скопируйте ссылку:Откройте необходимый файл списка (например, whitelist.txt) в этом репозитории.В правом верхнем углу панели файла нажмите кнопку Raw.Скопируйте URL-адрес из адресной строки браузера.
2. Добавьте в свой клиент:Sing-box / Xray / v2ray: Добавьте скопированную ссылку в блок удаленных правил (remote rules) или настроек маршрутизации.
